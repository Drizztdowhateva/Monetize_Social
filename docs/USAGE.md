# Usage

This file provides usage guidance for MonetizeSocial.

## Quick Start
1. Complete installation steps in docs/INSTALL.md.
2. Run the default command documented in the root README.

## Common Workflows
- Start the application or CLI.
- Run core features in a local test mode first when available.
- Review output logs and reports generated by the project.
- Open the GUI `Procedure` tab and run the Sponsorship + Affiliate Playbook for brand outreach and deal execution.
- Track sponsor leads and follow-ups in `data/templates/outreach_crm_lite.csv`.
- Use affiliate master data to prioritize sponsorship targets by niche fit and payout potential.
- Locate sponsor forms and qualification rules in `data/templates/sponsor_discovery_tracker.csv` before outreach.
- Add estimated investment amounts and set GUI sponsor budget to filter opportunities that are feasible now.
- Assign each sponsor to a vertical and use score-based ranking in the GUI to work top opportunities first.
- After ranking sponsors, run `Top Sponsors -> Hashtags` in the Hashtag Ads tab before generating captions.
- Use `Top Sponsors -> Captions` for one-click first drafts tied to ranked sponsor priorities.
- Run `Auto-Discover URLs` in Sponsors to prefill partner pages/forms and confidence notes.
- Run `Export Weekly Bundle` to generate follow-ups, attribution pack, ROI model, kanban board, and dashboard summaries.

## Troubleshooting
- Verify required dependencies are installed.
- Recheck configuration files and environment variables.
- Consult the root README and docs folder for advanced guides.
