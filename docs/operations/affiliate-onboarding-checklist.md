# Affiliate Onboarding Checklist

- Confirm niche fit and audience intent
- Verify program status and restrictions
- Submit application with approved website/channel
- Record affiliate ID and tags in tracker
- Configure payment profile and tax forms
- Add disclosure text to pages and videos
- Build first compliant links
- Launch first content and log campaign ID
- Review clicks/conversions after 7 days
- Optimize landing pages and call-to-action text
