# Easiest Affiliate Onboarding Playbook

This playbook is optimized for speed while staying compliant.

## 1. Start With Easy-Approval Programs

Prioritize:
- Amazon Associates
- ShareASale
- ClickBank
- FlexOffers
- CJ Affiliate

## 2. Prepare One Reusable Application Pack

Keep these ready:
- Website URL
- About page URL
- Contact page URL
- Primary niche statement
- Traffic/source summary
- Disclosure URL

## 3. Use the 15-Minute Workflow Per Program

1. Open row in `affiliate_master_fillable.csv`.
2. Click apply URL and submit application.
3. Update:
- `Application Submitted (Y/N)`
- `Application Date`
- `Approval Status`
- `Next Action`
- `Next Follow-up Date`
4. If approved, record:
- `Affiliate ID / Publisher ID`
- `Store ID / Associate Tag`
- `Tax Form Submitted (Y/N)`
- `Payment Profile Completed (Y/N)`

## 4. Tax Setup for Your Structure

For your setup (`Single-Member LLC taxed as Sole Proprietor`):
- Submit W-9 where required.
- Track all 1099-NEC or similar forms.
- Reconcile payouts monthly in `taxes_tracker_llc_1099_sole_prop.csv`.
- Track deductible costs (software, hosting, ads, equipment) for year-end filing.

## 5. Weekly Cadence

- Monday: apply to 3 new programs
- Wednesday: follow up pending applications
- Friday: reconciliation + compliance check
