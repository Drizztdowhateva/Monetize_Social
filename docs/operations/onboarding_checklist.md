# Affiliate Onboarding Checklist

## Entity Setup

- [ ] Legal entity name finalized
- [ ] Business address and contact email ready
- [ ] Tax profile prepared (W-9/W-8BEN or local equivalent)

## Publishing Assets

- [ ] Website or channel URL ready
- [ ] About page completed
- [ ] Contact page completed
- [ ] Disclosure page published
- [ ] Privacy policy published
- [ ] Terms page published

## Program Application

- [ ] Program requirements reviewed
- [ ] Application submitted
- [ ] Approval status tracked in CSV/XLSX
- [ ] Affiliate ID/tag recorded

## Commercial Setup

- [ ] Payment profile completed
- [ ] Payout threshold reviewed
- [ ] Promo assets and deep links collected
- [ ] Compliance restrictions documented
