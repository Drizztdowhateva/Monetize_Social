# Tax and Payout Checklist

## Tax

- [ ] Tax form submitted for each network/program
- [ ] Legal entity name matches payout account
- [ ] Country-specific tax obligations reviewed

## Payout Configuration

- [ ] Bank/PayPal details verified
- [ ] Payout currency confirmed
- [ ] Minimum payout threshold logged
- [ ] Expected payout schedule documented

## Reconciliation

- [ ] Monthly payout report exported
- [ ] Commissions matched to clicks/conversions
- [ ] Disputes filed before deadlines
